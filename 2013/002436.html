Hi there!

Welcome. I assume you have a computer you want to install Sorcerer on and
have already downloaded the ISO? Unfortunately we don't have much
documentation, as it's too much of a moving target. However, there is
plenty of help built into the ISO, including the deployment docet you are
asking about.

In general terms you want to boot off the ISO, make a place for the OS to
go, transfer it over, and then boot it.
I hope you are not planning on dual-booting, as that is generally only
lightly explored.. I have one machine that dual boots, so it is possible,
but it is not covered in the install menu, as far as I know. Unfortunately
for you at this stage, we don't install Sorcerer much, as it is a
continuously updating distro. I have had one install running for a few
years before installing from fresh late last year.

Once you have a clean boot of sorcerer, just type sorcery as root. It
brings up a menu where you can pick software to install. Once you have
picked all the software, execute the "cast" queue. Your requested software
will be downloaded in source form, and compiled for your machine.

Once you have a proper desktop going, we can show you how to play with the
internals of the system.

While the above sounds easy, you might hit a snag or two. Feel free to mail
us with a description of the issue. I can normally be reached on gmail
chat, but fortunately for me and unfortunately for you... I am spending
this week travelling home for some holiday. In general, software is
downloaded and compiled unmodified from the author's site, so if you hit a
snag with any software, the homepage of that software is the best place to
go.

:)

Kind regards,
Evert
