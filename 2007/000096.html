I don't use any desktop manager - it's a simple mysql server blackbox sorta
system.

BTW - I found what was making sticky eth devices... there's a
udev_write_rules thing that runs that makes a file in
/etc/udev/rules/70-static-network-rules (or something like that) which lists
all previous eth devices, and their mac addresses.  Removing this file and
rebooting solved having my first eth device at eth3.
