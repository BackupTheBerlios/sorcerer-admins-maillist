So far after having installed it the X server has quit responding
once already while using 100% of the CPU.

As I was writing this message it locked up again.

My first solution thus far was to re-install rovclock and reduce the
clock rate of the GPU from 300 to 100.

My second solution was to disable AGP 3.0 calibration and fast writes
in BIOS CMOS.


Be aware that if you do install the pre-compiled binary drivers
provided by ATI that there is no guarantee that it will not
melt your GPU's fan
In the past I melted 2 ATI GPU fans.
