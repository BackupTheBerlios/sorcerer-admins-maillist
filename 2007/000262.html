Reads like a worth request for enhanced augur functionality?


source         dir         Copy sources from dir into the source cache
source         file        Copy file             into the source cache
source         spell       Print the sources for the spell


The convenience of not having to specify the spell or version comes at a cost.
Sources to be copied into the source cache must be
a source that is owned by a current version of a spell.
Otherwise, augur would be unable to determine where the file should go.

Now to test it.
