Try that IP address.  :)

If there is a DNS problem it may be cached for a while
longer than it exists.
When trying to overcome a DNS problem always try
to ask directly, using dig, the name server that the
GTLD server says is the authority for the domain.

That way it skips all the caching name servers in between that
may still have the negative response cached.
