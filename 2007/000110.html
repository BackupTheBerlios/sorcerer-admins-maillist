I can have NVIDIA-Linux-x86 provide MesaLib.
I have no idea if it is sufficiently complete.
Not sure what will happen if on a non NVIDIA chipset box
if NVIDIA-Linux-x86 is installed instead of MesaLib?
It might fail or it might provide a slightly faster implementation
or it might cause a large amount of installed software to recompile
if the SA accidentally selects NVIDIA-Linux-x86 on an existing installation.

There may be, in the grimoire, a spell for another somewhat more stripped
down Mesa installation that is faster, while being mostly software,
but may not allow all OpenGL programs to run correctly.
If it is still in the grimoire I do not know.
Nor do I recall the name.
I made the change to NVIDIA-Linux-x86 to EPROVIDE MesaLib.
Hope that helps.
