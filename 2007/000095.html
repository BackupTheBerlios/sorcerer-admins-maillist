On 6/2/07, J Decker <d3ck0r at gmail.com> wrote:
> The reason I was attempting to install hotplug.
>
> USB  drives - I would like to automount them upon plugin, and run a
> script... can bare udev handle that?

It should.
You need only write some rules for it.
You could add them to:

/etc/udev/rules.d/1st.rules

Also if you run kde as your window manager it may
detect the drives when you connect them
and then prompt you to mount them
if you make an entry for their mount points in /etc/fstab
and users are permitted to mount them.
