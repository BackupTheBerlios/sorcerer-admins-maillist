> Here is something useful for those wanting to run an emulator.
> If you have a simple disk file.
> You may want to be able to mount it through the loopback device
> in order to transfer files to or from it quickly instead of using HTTP 
or SSH.
> # mount -o 32256 /dev/loop0 img.2G
> # mount /dev/loop0
> When finished.
> # umount /dev/loop0
> # losetup -d /dev/loop0
> 
> I am uncertain if the offset is the same with vmware
> as it is with qemu's disk image format.
