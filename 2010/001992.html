Please be aware that # augur forget
is now different than it was before.
It is more complicated.
Forgetting what augur forget does is not difficult to forget.

forget     aron spells     *      selections about spells
forget     conf spells     *      configurations for spells
forget     opts spells     *      spells' optionals and config
forget     prov spells     *      where spell provides other spells

Please see the

# augur help forget

for a verbose description for which each type of forget does.
