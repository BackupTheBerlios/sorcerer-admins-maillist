Hi Nathan!

Thanks for sticking with this great distro. There's a lot that I still
need to learn about basic administration the Sorcerer way, but I am
hobbling together a doc w/photos for Plain Jane installations (Great Sammy
Hagar song by the way). Since it includes about 40 or 50 photos so far, I
can't just post it to the list so I'll post a link when I get it in enough
of an order to put up so you can go over it.

When I do, however, First wait for Kyle and the other experienced Sorcerer
admins here on the list pick it apart for accuracies/inaccuracies before
taking it as gospel though, to save you the hassle of more grief.

After about a day or so of posting the alpha doc, There should have been
sufficient time for the 'Sorcerers' to offer up some tips and pointers for
cleaning it up so it is at least usable.

I plan for a very comprehensive and continually growing doc over the next
few months, but like I said, I've still got a lot to learn about the
correct way to admin Sorcerer - i.e., managing init-scripts, since they
are dealt with unlike any other UNIX implementation.

It won't be pretty to start off with, but adequate enough for usage as a
general guide.

One thing though - you should familiarize yourself with cfdisk, coz I
don't use it, and the instructions will focus on fdisk (Might as well be a
purist when it comes to that too ;) ).
