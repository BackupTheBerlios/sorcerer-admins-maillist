The problem with: /sbin/sinstall  is that it tries to access fd0.

Reason:

Function: list_part
Is missing the line in the sed part:

\,^/dev/fd0,d


And this it is safe to assume that no one use a floppy to install sorcerer
from (or the at least going to have a lot of them ;))

2. ok
3. select device
4. select Device
