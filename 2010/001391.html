Just to possibly eliminate that particular concern, have you tried the
following:

echo "quit" | /path/to/grub --batch --read-only
--device-map=/tmp/grub.devices.map >/dev/null 2>&1

That should generate /tmp/grub.devices.map if it doesn't exist.  The
format per line is \(GRUBDEV\)\s+LINUXDEV, so awk can chew through it
pretty easily.
