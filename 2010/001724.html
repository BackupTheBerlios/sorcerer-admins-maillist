here is the way I do it :
cd /etc/networks.d
cp templates/links/generic eth-dhcp
vi eth/dhcp
seek for the line with IF
edit it so that it looks like  IF=eth0
keep all the other lines commented
:wq
/etc/init.d/network start

I personally prefer to do a network since I start some other service in my
case but if you just want dhcp

be careful that dhclient or dhcpcd might not work according to your dhcp
server... if it's not working try chmod -x $(which dhclient)
