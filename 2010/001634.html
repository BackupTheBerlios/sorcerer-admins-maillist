prior to mounting the filesystem.

Ah, speaking of installs...
Are there special tools for BTRFS filesystems?
Should the menu driven installer support the
installation onto a BTRFS filesystem?

BTRFS is a totally experimental filesystem
that might break on kernel updates.
The tools are only available from debian's pool
or their git repository.
Why do you want to use something so new and dangerous?

I forget how long would be required to create a new one,
perhaps a couple of days.
If I go for it then I/Rs might not be available this week.
Not sure if it is worth the wait since people who use it
might end up losing data.
