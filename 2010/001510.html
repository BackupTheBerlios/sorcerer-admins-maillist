Unless there is something tricky about it
then the procedure is straight forward.
First disable any boot/sector MBR virus protection
that is enabled in the BIOS/CMOS
The BIOS/CMOS can be reached on most computers
by pressing delete or F1 or escape while the computer
is POST, power on self test.

If Sorcerer is already installed then
the program ms-sys can be used to write
a bootable MBR to the master boot record.
When a computer boots the code at the front of the disk
is loaded and executed.
This area is called the MBR, master boot record.

lilo, the linux loader, should be installed either in the boot block
of the /boot partition or the / partition.
That partition that contains lilo must be marked as bootable
in the partition table by using a program such as fdisk.

Now if no other partitions are desired on a disk.
Then partitions that are not desired can be removed.
Alternatively, for a fresh install
one could remove all partitions from a disk
and then create just 3 partitions such as
a 1GB /boot partition
a 2GB swap partition
and allocate the rest of the disk space to the / partition.
It is a simple layout, but it works.

Please remember to backup the data from your fixed disk
prior to repartitioning it.
Any data loss due to following these instructions will not
be compensated since I do not have a backup of your data.  :)
Good luck.
