Not exactly tiny, not exactly perfect,
but still better than the old.
The new I/Rs are available for download.
Both have passed an install and reboot to installed system test.
This is the first time in years that the pre-compiled kernel on the I/Rs
is still current with the most recent stable kernel release as reported
on http://www.kernel.org/  :)
It is not yet obsolete.

When doing new installs please use the new I/Rs and not the older
one's that also have Xorg, samba, xine, etc... on them.

I tried using a LZMA compressed initramfs.
The kernel was never able to uncompress it.
So I settled on bzip2 compression of the initramfs.
Unfortunately, the decompression time is not fast as gzip.
So expect to wait a bit longer for the initrmafs to decompress.
The initramfs decompress time only applies to booting with the I/R
initramfs created on installed boxes are not compressed.
