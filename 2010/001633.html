I do use the BTRFS file system on my home computer.

I will actually have to give you a copy of some an init file I made. When
you use BTRFS for a raided system, you have to scan the disks before
mounting the file systems.

I have two 1TB hard drives that I have combined in a RAID 0 config with
BTRFS. It's amazingly fast. Also, it's easy to add and remove storage on a
BTRFS file system.

As long as it's possible to tune an installed modular kernel to support
BTRFS, it should be fine.
