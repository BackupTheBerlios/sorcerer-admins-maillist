Then your /etc/fstab could look like this:

/dev/hdc1        /boot   ext2 defaults,noatime 0 1
/dev/hdc2        /         xfs    defaults,noatime 0 1
/dev/hdc3        swap   swap  defaults    0 1

If your installation uses initramfs then after modifying the /etc/fstab config 
file you still need to recreate the initramfs image so the new /etc/fstab is 
used during boot.
