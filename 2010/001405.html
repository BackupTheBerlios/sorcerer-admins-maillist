Hi there, Nathan.

It's nice to see people join Sorcerer. To  be honest with you, installing
Sorcerer is not one of the easiest tasks as once Sorcerer is installed it
tends to stay that way. I install Sorcerer quite often, and the last time I
did a fresh install was about a year ago.

Nevertheless, I would like to help. Can you describe to me where it is that
you are getting stuck?

Sorcerer install can basically be broken up into a few seperate steps.

1. Sorcerer I/R disk boots, and gives you a prompt.
2. The installer is excuted.
3. Partitions are made
4. Files are transferred
5. MBR is updated ( optional )
6. Reboot into Sorcerer.


Can you describe to me which one of these is giving you a problem, please?
