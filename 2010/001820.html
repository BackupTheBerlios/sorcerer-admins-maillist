> You don't need to use UUIDs in /etc/fstab. Instead you could use, for example:
> 
> /dev/mapper/ssd-root / ext4 defaults,commit=600,noatime 0 1
> /dev/sda1 /boot ext3 defaults,commit=600,noatime 0 1
> 
> So your boot partition would need a line like:
> 
> /dev/hdc1 /boot ext3 defaults,noatime 0 1
> 
> You obviously need to adjust the names and options according to your system.
