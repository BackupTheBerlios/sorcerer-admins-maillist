Well currently the 64 bit version is, as I see it, not for the regular
user...sorry Microsoft. The reason for that is, that not all manufatures has
released final 64 bit versions of their drivers and their software. But
Microsoft is on other hand forced to move to 64bit - if they want to stay in
the "game" in the longer time-range, as the Chip-producers most likely will
stop supporting 32 bit emmulation on 64 bit in a "shortere" timeframe (2-4
years).

So if one has to get Vista with a new machine - then currently the 32 bit is
the way to go right now.  But nothing is stopping you from installing any
other OS like Linux.
