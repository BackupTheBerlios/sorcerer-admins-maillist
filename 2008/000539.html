> "Why is it re-casting $SPELL at the same version as is installed?"
> I removed newer_ia if for no other reason
> than to eliminate those type of email questions
> from new Sorcerer SAs.
> 
> To execute newer_ia,
> run augur newer
> then find all files in the cast queue
> then inside a while read; do ...; done loop
> echo "$REPLY $( augur version $REPLY ) $( augur available $REPLY )"
> then after the done you can pipe it through column
> in order to make it look pretty.
> 
> Install SA made augur command modules in
> /usr/local/libexec/sorcery/augur.d/
> 
> After writing a newer_ia  augur command module
> writing any augur command modules will be easy  :)
> /usr/libexec/sorcery/augur.d/
> contains the augur command modules
> that come complimentary with sorcery.
> 
> 
> Eventually, augur should undergo further revision
> to ensure that it has a proper set of basic building
> blocks for Sorcerer SAs
