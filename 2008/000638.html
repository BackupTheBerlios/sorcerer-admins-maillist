New command summary:

latest     install         View the available versions for installed spells
latest     none            View the available versions for all spells
latest     spells          View the available versions for spells
secure     none            View the secure versions for installed spells
secure     spells          View the secure versions for spells
stable     none            View the stable versions for installed spells
stable     spells          View the stable versions for spells
unstable   none            View the unstable versions for installed spells
unstable   spells          View the unstable versions for spells


# augur latest

The above outputs in 3 text columns.
Due to large spell names and large versions
it can exceed the standard 80 character column wide terminal emulation.
However, latest can be used with specific spells named
or with the "install" parameter instead to list information
only about spells that are already installed.

# augur available

is unchanged.
It still prints the available version based upon select version
progress for that spell.
Therefore, it could print the version which is:
stable, unstable, secure, or the version which the spell is being held at.
