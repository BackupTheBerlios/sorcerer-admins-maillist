wine may not run properly or even compile on x86_64
since windows, as far as I know, including windows vista
is a 32 bit operating system.
If it were me I would install Sorcerer IA32 in a directory
and then update it regularly and run wine from there.
The x86_64 linux-modular kernel supports ia32 emulation.
However, do not mix ia32 and x86_64 in the root filesystem.
Suggest x86_64 for the root filesystem and
ia32 for a chrooted filesystem.

If anyone runs wine on x86_64, please correct what I wrote above.
