Years ago someone requested for a portable leech
that is detached from requiring a grimoire and sorcery.

Here is the augur command to generate it:

minileech  spells          create script that downloads sources for spells

It will output the name of the script it creates.
The script can be executed on any box with BASH and CURL installed.
It will create ~/sources/$SPELL/$VERSION/$SOURCES
Those can be transported to somewhere else via a zip drive, CDROM,
flash drive, etc...

The minileech script does not:
download source patches from the PPR,
download sources from Bent's source cache repository,
check that the files downloaded are sources and not XML's with some
fancy text saying 404,
check that file download is complete,
check integrity of compressed files,
etc....

It is about as dumb and lame as a download program can be.
However, it might just work for 95%+ of source downloads.
Recent changes in sorcery make writing a minileech script easier.

Since minileech performs no checks,
copying downloads into a source cache
may cause cast failures when those sources
are corrupt, incomplete, or not really a source.
