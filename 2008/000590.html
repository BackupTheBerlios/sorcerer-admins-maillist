It will spew many errors about:
stable unstable and secure do not exist.
That is ok.
The errors go away after a sorcery compatible with the grimoire is installed.

# augur sync
# augur index
# augur newer
# augur update


The post_install for the newer sorcery should also
convert hold sold and exile logs into progress logs.

However, if in order to use a unstable/developmental version of an
xorg video driver
please issue the request using

# augur progress $SPELL unstable
