mdadm           75% complete, 5% tested, not in installer yet

1 install test deployment with cryptsetup LUKS and reboot test.
1 install test with mdadm RAID and reboot test.
1 install test with LVM2 and reboot test.

Sorry for the delay.
Except for miscompiling the installed software to athlon-xp
architecture the 20080501 ia32 I/R is probably nearly bug free.

any potential of lurking athlon-xp only executables.
If you want it urgently I can roll a beta of it after
completing the mdadm support in the installer.
