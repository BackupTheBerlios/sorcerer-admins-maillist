The sorcery menu for editing configuration files in /etc is gone.
You probably never used it, right?

Want to adjust the default PATH on a box,
but not certain what file in /etc to edit?

# cast -r --deps etc

Simply select the file to edit from the etc spell's configuration menu.
In that fashion the default PATH can be changed, the prompt, and more.

The etc spell is a meager 2K.
It works like a mini sorcery.
Files in etc.d/ are like mini spells.
They specify what can be changed
what it does, and how to change it.

This way adding new entries to etc spell's configuration menu
is as easy as adding new files to etc.d/
If you make new entries
then please tar gzip them and send them my way.
Then, I can add them into grimoire/etc.d/

This change accomplishes achieving greater functionality
while at the same time greatly mitigating the amount of damage
that an error can cause, because
a problem in a sourced in file for the etc spell
will probably not break the etc spell and
will definitely not break augur, cast, dispel, leech, and sorcery.
