Hey i am not here asking grimoire-stable or sorcerer-stable, dont get mad 
at me :) 

IMHO using always latest version available you are in bleeding edge, you 
never know whats new version is going to do if you dont test it before 
installing it, right?
Its very true that new versions has bug fixed but also it goes backwards, 
new versions come with new bugs, am i right?

You all know that major distros do actually freezes versions when they are 
releasing new distro version, but its easier with binary distros because 
majority is using precompiled binaries and with package manager's 
dependencies with on it. So its magnity harder with source based distros 
like Sorcerer is to maintain same level of stability. Allthough you can 
argue stability like in Ubuntu when they do take packages from Debian 
unstable (?) repository, but something like Debian main releases tend to 
be rock solid. 

Also dont forget security fixes, if grimoire is freezed then either we 
backport security fixes or release spell with version that has security 
bug fixed and that could mean it has some new dependencies regarding other 
spells, bugs, configuration changes, etc.
