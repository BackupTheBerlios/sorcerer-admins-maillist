For all you snaky fans Python 3.0 will be added as a latest in version progress.
Sources that require Python and can not detect Python 3.0 exist.
Short of recompiling the entire grimoire they are troublesome to locate.
Therefore, Python 3.0 will not be rushed to stable.
However, if you do not mind sediting the breaks
then feel free to

# augur progress Python latest

Just because a version is made available as unstable or latest or secure does
not prevent the SA from selecting it.
Feel free to go with the version that works best for you.
That is the Sorcerer way, of course.
