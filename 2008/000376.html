On Jan 27, 2008 4:39 AM, jean-luc malet <jeanluc.malet at gmail.com> wrote:

> sorcery provide system hooks, for example

Hooks can be defined in /etc/sorcery/local.config

Available cast hooks:

hook_pre_build()
hook_build()
hook_post_build()
hook_prepare_install()
hook_post_install()

Available dispel hooks:

hook_pre_remove()
hook_post_remove()

hook_*() do not replace the non hook_ functions.
hook_*() execute before the expected function.

Example:

hook_pre_build() executes before
pre_build()

The return value of hook_*() is discarded.

This overly simple implementation
required the least modification
and the least additional complexity.
Perhaps it will suffice?
Please try it.

Available in
sorcery version implosion-20070127-1
and later.
