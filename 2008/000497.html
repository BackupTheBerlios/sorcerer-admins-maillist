Next version of module-init-tools will install /usr/sbin/{modprobe,rmmod}
They are wrappers for /sbin/{modprobe,rmmod}, respectively.
When invoked with a single parameter
the wrappers will prompt, "Also load/unload during boot?"
Press y or Y and /etc/modules or /etc/modules.rm will be updated.

The next update of xorg-server will add an init-script
that checks for /etc/X11/xorg.conf and
if /etc/X11/xorg.conf is not present
then execute X -configure
and install a potentially useful /etc/X11/xorg.conf

Commonly users add ~/bin to the front of PATH
With the next update of bash ~/bin will be added to the
front of PATH when $HOME is defined.

Those conveniences,
which are more useful on new boxes than deployed boxes,
alone are insufficient reason to compel
those installed spells to recast automatically.
When definitely desired on deployed boxes
please cast the spells after the installed grimoire
becomes version 2008-163 or later.
