What files are missing?
Files in:

/usr/doc
/usr/share/doc
/usr/share/locale/*/LC_MESSAGES
/usr/src/linux

and some other directories are not added onto the I/Rs
They can be regenerated when the spells are re-cast.
They are omitted because translations and docs that
are seldom read can add on another 50M to 100M
in the compressed size of the I/R

It may be that during installation the CDROM had read errors
which caused some files to not transfer or to transfer corrupted.

Before the I/Rs are rolled
sorcery-md5sum is cast.
That verifies that all the essential files
for which md5sums exist are present,
with the exception of some files in /etc

The I/Rs should have all the files required to deploy
a nearly base Sorcerer box.
