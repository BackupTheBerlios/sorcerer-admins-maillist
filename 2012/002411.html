The paradigm feature is gone.
I disliked the implementation.
Consequently, sorcery no longer casts spells in paradigm mode.

However, sorcery can run chrooted.
Consequently, many of the functions provided tacitly
are now provided as verbs for the augur paradigm module.
Paradigms can be created under /media/root/
with a name specified by the SA.
Paradigms can be relinked with
/ to reduce space consumption.
/ can be shifted to a paradigm.
Consequently, the functionality remains present,
but the previous implementation as a feature is removed.
Any paradigm stored within /para/
can be moved to /media/root/
without requiring additional editing.

For a verbose description of paradigms,
what they are, how to use paradigms effectively, etc...
please
# augur help paradigm
