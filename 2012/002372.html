Hi there, Demetri.

In your case I would recommend a virtual machine. To move a virtual machine
to real hardware is not too difficult, and I can help you with that when
the time comes.

The virtual machine has some very real advantages. It does not touch your
running OS, and it's virtual hardware is quite predictable.
In this way you will be able to get a "feel" for how Sorcery operates, and
once you are confident enough, you can try dual-booting.

I can help you with the dual boot too, and with the migration to real
hardware.

However, for now I recommend using a virtual machine.

Happy hacking!
