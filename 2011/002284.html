# /etc/init.d/tomoyo-userpass configure

Running the above command allows for adding and removing
permission for a user to enter the freedom domain by running /domain/freedom
Rather than accepting a single password
it asks for a password to be associated with a user account.
Therefore, each user with permission to enter /domain/freedom
will have their own password which should be separate from
the user's login password.
