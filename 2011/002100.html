Kyle,

There are a number of people coming into the channel asking about the
404 errors on the website and how/where they can find Sorcerer and the
related files. etc.

Fully half of all the places I've been on the net with info on Sorcerer
point to http://sorcerer.aakin.net

We need to change this in the IRC chan to the correct link. Can you log
on to chan #sorcerer on Freenode and change the topic, and preferably
set 'tallship' and 'grimdin' +* with the following  commands?:

<code>
/msg chanserv flags #channel tallship +*
/msg chanserv flags #channel grimdin +*
</code>

That will confer all neccessary chanop privs and abilities except for
Founder status.

NOTE: As founder, you will need to affect the changes above yourself.

Here's the info on the chan from chanserv:

<code>
Information on #Sorcerer
Founder		: Freenode staff
Registered	: Oct 12 02:13:07 2001 (9 years, 23 weeks, 4 days, 18:00:42 ago)
Last used	: Oct 22 14:14:56 2008 (2 years, 21 weeks, 6 days, 05:58:53 ago)
Mode lock	: +nt-s
URL		: http://sorcerer.wox.org
Email		: cromwell at kublai.com
Flags		: KEEPTOPIC TOPICLOCK
***End of info***
</code>

I recall you saying you think you've forgotten how to use IRC, so I've
got a little primer here for you to follow if need be. I'll type if out
first person so it will be pretty much just a cut and paste for you, if
you're so inclined to fix this.

Grimdin and I are more than happy to idle and maintain the chan for you :)

Here goes.

<code>
$ irssi
/connect -ssl chat.freenode.net
/nick cromwell-tmp
/msg nickserv recover cromwell 'password'
/msg nickserv release cromwell 'password'
/nick tallship
/msg nickserv identify 'password'
/msg chanserv recover #sorcerer
/msg chanserv flags #channel tallship +*
/msg chanserv flags #channel grimdin +*
/join #sorcerer
Hello you two little script kiddies!
/part
/quit
$
</code>

You'll have to supply the password yourself ;), yet in the case it has
been forgotten, Freenode does have pwd recovery facilities.

Can you give this a shot?

If so, I can build a helpbot to put in there which will help maintain
the chan and can answer questions specific to Sorcerer, etc. Could be a
good thing for command syntax and FAQ material, etc.
