If access to the root file system is possible

then

[ -d /etc/sorcery/ ]

Some distros put a file in /etc/
that indicate the distro and release version of the Install CD set.
However, such information is meaningless with Sorcerer.
The presence of the directory /etc/sorcery/ suffices.

Usually from examining services or doing a port scan
there is nothing that brands a box as Sorcerer.
Nothing about a Sorcerer box
creates the facade that it is any specific distro.
Sorcerer is a moderately stealthy distro.
However, the SA can further increase stealth.
