I tried running with the newer xorg-server, twice...

nVidia does not like the newer server, and probably won't release a
compatible driver until the newer server is actually used by a few more
people. I guess we will have to wait for the server to be used by Ubuntu or
such.
The error that I am getting is a module mismatch. I have actually tried
compiling xf86-video-nv, but this fails to compile with xorg-server 1.7.0

I know for a fact that nVidia binaries work with xorg-server 1.6.2.

The latest Ubuntu uses xorg-server 1.6.4, which has a good chance of being
supported by the nVidia drivers.

Our challenge here is to find a combination of packages that will allow
xorg-server 1.6.4 to compile.
