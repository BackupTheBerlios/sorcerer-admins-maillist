vurl is short for version url.
Sometimes neither the url source nor homepage gives a clue
as to what the most recently tarball.
Therefore, when vurl is present
vurl is checked to see if a new tarball was released.
See how the download URL contains the version number.
Strip off the filename.
Look in that directory.
Will you ever see the version 1.0.5 tarball?
Probably not.
Therefore, a vurl is required to indicate
where to look in order to receive notice
that a version 1.0.5 tarball was released.
