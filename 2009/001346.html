Some spells could have a certain version set
based upon a case statement
if NVIDIA-* is detected as installed.
However, this only keeps existing systems from breaking.
It does not make for easy installation of NVIDIA-*
on a new box.

Therefore, I suggest that the legacy version progress
be used to hold a the versions for the set of xorg spells
that are compatible with the NVIDIA-* and the ATI-*
pre-compiled binary drivers.

There is no easy solution to it.
Probably, if I were considering the change
then I would stick with the set of xorg spells
that work for xorg, NVIDIA and ATI drivers.

However, at least if those versions are available
as legacy versions
then making the change to legacy versions
will not be difficult.
